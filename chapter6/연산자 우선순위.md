## 연산자 우선순위

우선순위가 높은 연산자가 먼저 계산되고, 우선순위가 같으면 좌측부터 우측으로 계산된다

| 우선순위 | 연산자               |
|------|-------------------|
| 5    | * / % << >> %& &^ |
| 4    | + - \|  ^         |
| 3    | == != < <= > >=   |
| 2    | &&                |
| 1    | \|\|              |

### 3 * 4 ^ 7 << 2 + 3 * 5 == 7 의 경우

볼드체의 연산이 순서대로 계산된다.

**3 * 4** ^ 7 << 2 + 3 * 5 == 7

12 ^ **7 << 2** + 3 * 5 == 7

12 ^ 28 + **3 * 5** == 7

**12 ^ 28** + 15 == 7

**16 + 15** == 7

31 == 7
-> false
